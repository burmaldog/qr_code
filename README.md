# QR Studio

**QR Studio** is a sleek, high-performance web application designed for instant generation and customization of matrix barcodes (QR codes). Built with a focus on speed, privacy, and professional-grade output, it provides a seamless experience for both casual users and designers.

---

## ✨ Key Features

- **⚡ Real-Time Generation**: Watch your QR code evolve instantly as you type. No reload or "Generate" button required.
- **🛠 Advanced Customization**:
    - **Error Correction Levels**: Choose from four levels (Low, Medium, Quartile, High) to ensure your code remains scannable even if partially damaged or obscured.
    - **Multiple Formats**: Export your results in **SVG** (Scalable Vector Graphics) for professional design work or **PNG** for quick social media sharing.
    - **Pro Resolutions**: Generate high-fidelity images ranging from **512px** up to **4096px**, perfect for everything from business cards to giant billboards.
- **📜 Smart History**: Automatically tracks your last 5 unique generations. Switch between recent inputs with a single click without re-typing.
- **🎨 Premium UI/UX**: A minimalist "Apple-style" interface featuring **Glassmorphism** effects, smooth transitions, and a fully responsive layout that works beautifully on mobile and desktop.
- **🔒 Privacy First**: All processing happens locally in your browser. Your data never leaves your device and is never sent to a server.

---

## 🚀 How It Works

Using **QR Studio** is straightforward and efficient:

1. **Enter Content**: Type or paste your URL, text, or data into the main input field.
2. **Configure Settings**: 
    - Select the **Error Correction** level based on your needs.
    - Choose your preferred **Export Format** (SVG is best for printing).
    - Pick a **Resolution** (higher is better for large-scale use).
3. **Preview**: View the generated code in the dynamic preview card. The "Glassmorphism" card displays the data and creation timestamp.
4. **Download**: Hit the "Download" button to save the file directly to your device.

## 🛠 Tech Highlights

*   **Frontend**: Tailwind CSS v4 for ultra-fast, modern styling.
*   **Logic**: Pure JavaScript (ES6+) for a lightweight and responsive experience.
*   **Engine**: High-performance client-side QR synthesis library.
*   **Animations**: Custom CSS-based motion for a fluid feel without heavy external dependencies.

## 📦 Quick Start

Since **QR Studio** is a client-side application, you don't need to install any servers or dependencies.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/qr-studio.git
   ```
2. Open `index.html` in any modern web browser.
3. Start generating!
